Operations Center — Tool Selection

Select a tool below to begin. All operations run locally — no data leaves this machine.

HTTP Header Scanner

Analyze security headers on target URLs. Checks HSTS, CSP, X-Frame-Options, and more. Assigns a security grade.

Hash Toolkit

Generate MD5, SHA-1, SHA-256, SHA-512 hashes. Verify a hash against known input. Auto-detects algorithm by length.

Subnet Calculator

Enter CIDR notation to calculate network address, broadcast, usable hosts, first/last host, and subnet mask.

Base64 Codec

Encode and decode Base64 strings. Useful for analyzing obfuscated payloads, encoded credentials, and data blobs.

URL Decoder

Decode URL-encoded strings. Parses and displays scheme, host, path, and query parameters separately.

Log Analyzer

Paste log entries to extract IPs, emails, URLs, hashes, CVEs, error keywords, and other IOC patterns.
