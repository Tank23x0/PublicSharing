# Remote Work Security - Key Talking Points

## Why Remote Security Matters
- Home network is now an attack surface
- 47% increase in phishing targeting remote workers
- 20% of remote workers have experienced breach attempts
- Your security affects the entire organization

## Home Network Security

### WiFi Configuration
- Use WPA3 or WPA2 encryption (never WEP)
- Change default router password
- Create strong WiFi password
- Consider separate network for work devices
- Hide network name (SSID) if possible

### Router Security
- Update firmware regularly
- Disable remote management
- Change default admin credentials
- Enable router firewall
- Disable WPS (WiFi Protected Setup)

### Public WiFi
- Avoid for work if possible
- Always use VPN if you must
- Never access sensitive data without VPN
- Disable auto-connect to open networks

## Device Security

### Updates and Patches
- Enable automatic updates
- Don't postpone security updates
- Restart regularly to apply patches
- Update all software, not just OS

### Physical Security
- Don't leave devices unattended
- Lock screen when away (Win+L / Cmd+Ctrl+Q)
- Store securely when not in use
- Use cable locks for laptops

### Work/Personal Separation
- Use work devices for work only
- No family use of work equipment
- Keep accounts separate
- Don't store personal files on work devices

## VPN Usage

### Best Practices
- Connect before accessing company resources
- Stay connected during work
- Use only company-approved VPN
- Report connection issues to IT

### When VPN is Required
- Accessing internal systems
- Handling sensitive data
- Using any WiFi outside your secured home
- Always for public WiFi

## Physical Environment

### Screen Privacy
- Position monitor away from windows
- Be aware during video calls
- Use privacy screens in shared spaces
- Close sensitive windows before screen sharing

### Audio Privacy
- Use headphones for confidential calls
- Find private space for sensitive discussions
- Mute when not speaking
- Be mindful of smart speakers (Alexa, Google Home)

### Document Handling
- Shred sensitive printouts
- Lock physical documents
- Don't leave documents in shared areas
- Return confidential materials to office

## Video Conferencing Security
- Verify participants before sensitive discussions
- Use waiting rooms and passwords
- Don't share meeting links publicly
- Be aware of what's visible on camera/screen share
- Record only when necessary and with consent

## Remote Work Threats

### Increased Phishing Risk
- Fake IT support emails
- "VPN update" scams
- Credential harvesting
- Payment redirect attacks

### Social Engineering
- Attackers research home environments
- Impersonation of delivery services
- Phone-based social engineering
- Verification requests for "security updates"

## Incident Response from Home
- Report incidents immediately
- Keep IT Security contacts handy
- Know how to disconnect quickly
- Save IT number in your phone

## Key Statistics
- 47% increase in phishing during remote work
- 60% of breaches involve remote work vectors
- 68% of remote workers don't follow all security protocols
- $137 more per record for breaches involving remote work

## Key Principle
> "Your home office is now part of the corporate security perimeter. Same standards, same vigilance."
