# Physical Security - Key Talking Points

## Why Physical Security Matters
- Physical access bypasses digital defenses
- Stolen laptops expose thousands of records
- Hardware implants can persist undetected
- Dumpster diving yields valuable information
- Social engineering is easier in person

## Access Control

### Badge Security
- Wear badge visibly at all times
- Never lend your badge
- Report lost/stolen badges immediately
- One badge = one person

### Tailgating Prevention
- Don't hold doors for unknown people
- Politely ask for identification
- Direct strangers to reception
- Report tailgating attempts

### Visitor Management
- All visitors signed in and escorted
- Never leave visitors unattended
- Visitors should wear visible badges
- Sign visitors out when they leave

## Workspace Security

### Clean Desk Policy
- Lock away sensitive documents
- Clear desk at end of day
- No passwords on sticky notes
- Position monitors away from traffic

### Screen Security
- Lock computer when stepping away
  - Windows: Win+L
  - Mac: Ctrl+Cmd+Q
- Use privacy screens
- Be aware of shoulder surfing
- Close sensitive windows before meetings

### Document Handling
- Use secure print release
- Shred (don't trash) sensitive documents
- Don't leave printouts on shared printers
- Secure filing cabinets

## Device Security

### Laptops/Mobile
- Never leave unattended in public
- Use cable locks in office
- Store securely overnight
- Report lost/stolen immediately

### Removable Media
- Don't plug in unknown USBs
- Encrypt portable storage
- Track and account for media
- Securely wipe/destroy old drives

## Home Office Security
- Same standards as the office
- Secure devices when not in use
- Don't let family use work devices
- Shred sensitive printouts
- Lock doors during video calls

## What to Report
- Unfamiliar people in secure areas
- Propped doors or broken locks
- Tailgating attempts
- Missing equipment
- Unknown devices on systems
- Unusual questions about security

## Physical Security Zones
1. **Public** - Lobby, public areas
2. **Restricted** - Office spaces, requires badge
3. **Secure** - Server rooms, executive areas
4. **Highly Secure** - Data centers, vaults

## Key Principle
> "If you can touch it, you can own it. Protect physical access like you protect passwords."
